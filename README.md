# 币安合约标记价格监控系统 V3.0
一个实时监控币安合约标记价格的 Web 应用，支持跨浏览器、跨设备访问同一监控列表。

## ✨ 核心功能

- ✅ **实时监控**：通过 WebSocket 实时获取币安合约标记价格
- ✅ **价格警报**：价格超出设定范围时触发声音 + 弹窗警报
- ✅ **跨设备同步**：基于 SQLite 数据库，支持多设备访问同一监控数据
- ✅ **警报历史**：自动记录所有触发的警报，可查询和管理
- ✅ **响应式设计**：支持桌面和移动设备访问

## 🏗️ 技术架构

### 前端
- **框架**：原生 JavaScript + Bootstrap 5
- **实时数据**：WebSocket（币安 Mark Price Stream）
- **UI**：响应式设计，移动端适配

### 后端
- **服务器**：Node.js + Express
- **数据库**：SQLite（轻量级文件数据库）
- **API**：RESTful 接口

## 📦 安装步骤

### 1. 安装 Node.js
确保已安装 Node.js（建议 v14 或更高版本）。

### 2. 安装依赖
npm install


### 3. 启动服务器
npm start


### 4. 访问应用
在浏览器中打开：`http://localhost:10000`
